== Description ==

This plugin is add search button in plugin-install page.


== Installation ==

1. Upload `add-plugin-search-button.zip` to your plugins directory ( usally `/wp-content/plugins/` )
2. Unzip the `add-plugin-search-button.zip` file
3. Activate the plugin through the 'Plugins' menu in WordPress


== Screenshots ==

1. You will be able to click on the plugin search button.
